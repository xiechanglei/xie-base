# xie-base-common

引入依赖
```xml
<dependency>
    <groupId>io.github.xiechanglei</groupId>
    <artifactId>xie-base-common</artifactId>
    <version>2.7.17.2</version>
</dependency>
```

所有xie-base-common项目的合成模块，默认包含了以下模块：
